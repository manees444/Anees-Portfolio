# Muhammad Anees Portfolio Website

A premium AI-tech themed personal portfolio website for **Muhammad Anees** showcasing expertise in SEO, AI automation, digital marketing, and web execution.

## Live Website

Visit the live portfolio here:  
**[https://anees-portfolio-two.vercel.app/](https://anees-portfolio-two.vercel.app/)**

## About This Website

This portfolio is designed to present:

- Professional profile and positioning
- Experience timeline
- Skills and expertise categories
- Education and certifications
- Featured and detailed project work
- Contact and collaboration options

The site uses a modern dark glassmorphism interface with subtle motion, interactive components, and a mobile-first responsive layout.

## Tech Stack

- **HTML5**
- **CSS3** (custom styling + responsive breakpoints)
- **Vanilla JavaScript** (animations, filters, interactions, dynamic year)
- **Font Awesome** icons
- **DM Sans** + **Monaspace Neon** typography

## Project Structure

```text
.
├── index.html          # Homepage
├── styles.css          # Main global styling
├── projects.html       # Projects detail page
├── projects.css        # Projects page specific styling
└── images/             # Website/project assets
```

## Features

- Sticky glassmorphism navigation
- Animated hero background and typewriter effect
- Scroll-based reveal animations
- Skill bar animations and interactive timeline
- Project cards with modal-based detailed view
- Category filtering on projects page
- Dynamic footer year (`currentYear`) via JavaScript
- Fully responsive across mobile, tablet, and desktop

## Run Locally

Since this is a static website, you can run it directly:

1. Clone this repository
2. Open the project folder
3. Open `index.html` in your browser

Optional (recommended): serve with a local static server for best behavior.
